# Tech Stack

To ensure a reliable, modern, and high-performance experience, Agent Clinic will utilize the following technologies:

## Frontend
- **Framework**: React (TypeScript)
- **Styling**: Vanilla CSS (Modern, platform-native aesthetics)
- **State Management**: React Context or Zustand (if needed)

## Backend
- **Environment**: Node.js
- **Framework**: Next.js or Express (TypeScript)
- **Database**: PostgreSQL or SQLite (Reliable relational data storage)

## Testing & Quality
- **Unit/Integration**: Vitest or Jest
- **Linting**: ESLint
- **Type Safety**: TypeScript (Strict mode)
